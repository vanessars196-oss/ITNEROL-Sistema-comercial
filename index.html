TicketLog — Pedágio Mensal

Como usar o TicketLog:
  1. Exporte o relatório mensal de pedágio do sistema Ticket Log em CSV (separado por ; ou ,).
  2. Clique em Importar CSV — o sistema detecta automático as colunas de placa, prefixo, valor e mês.
  3. Registros são agrupados por veículo+mês (soma automática de múltiplas transações).
  4. O sistema vincula cada veículo ao contrato pelo prefixo cadastrado. Se não encontrar, clique Vincular Contratos.
  5. Na Apuração Mensal, ao selecionar Contrato + Mês, o pedágio é preenchido automaticamente do TicketLog.
Colunas aceitas: PLACA, PREFIXO (ou FROTA/NUM_FROTA), VALOR (ou TOTAL/PEDAGIO), MES_ANO (ou DATA/COMPETENCIA), QTD.

Relatórios & Backup de Dados

🔒 Segurança e Continuidade dos seus dados

Os dados ficam armazenados na plataforma GenSpark (banco de dados em nuvem). Para garantir que você nunca perca informações, exporte regularmente usando os botões abaixo.

CSV por módulo — Importável no Excel ou Google Sheets. Cada módulo tem seu botão de exportação.
Backup JSON completo — Exporta TODOS os dados em um único arquivo. Use para restaurar o sistema se necessário.
E se eu perder acesso ao GenSpark? Com o Backup JSON você mantém todos os dados. Um desenvolvedor consegue restaurar o sistema em outra plataforma usando esse arquivo. Recomendado: faça backup mensal.
